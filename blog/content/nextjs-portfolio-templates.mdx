### Controller

In your PaymentController, add functionality for checkout:

- **Important**: these properties uses a static payment information, change it according to your needs.

```php
 public function checkout(Request $request)
    {

        $order = BookingOrder::create([
            'reference' => uniqid("ORDER-"),
            'payment_id' => null,
            'product_id' => 1,
            'quantity' => 1,
            'status' => 'PENDING'
        ]);

        $response = Http::withBasicAuth(env('PAYMAYA_PUBLIC_KEY'), '')
            ->post('https://pg-sandbox.paymaya.com/checkout/v1/checkouts', [
                "totalAmount" => [
                    "value" => 10.00,
                    "currency" => "PHP"
                ],
                "buyer" => [
                    "firstName" => "John",
                    "lastName" => "Doe",
                    "email" => "john@example.com",
                    "contact" => [
                        "phone" => "+639171234567",
                        "email" => "john@example.com"
                    ]
                ],
                "items" => [
                    [
                        "name" => "Iphone 18 pro max",
                        "quantity" => 1,
                        "totalAmount" => [
                            "value" => 10.00,
                            "currency" => "PHP"
                        ]
                    ]
                ],
                "paymentMethods" => [
                    'card',
                    "paymaya",
                    "gcash",
                    "qrph"
                ],
                "redirectUrl" => [
                    "success" => route('payment.success'),
                    "failure" => route('payment.cancel'),
                    "cancel" => route('payment.cancel')
                ],
                "requestReferenceNumber" => $order->reference
            ]);

        $data = $response->json();

        // redirect user to Maya checkout page
        return redirect($data['redirectUrl']);
    }
```

### Adding Webhooks

After the setup is done and your webapp is accessible in public through ngrok

Open the Paymaya Dashboard once again:

1. Copy the url in ngrok
2. Go to Settings and select webhooks
3. Select which Merchant you selected before
4. if isnt empty, delete all existing url and replace yours
5. Hit on Test
6. Result should be 200 Ok, not 419 or 404 error code

If 419 error occured your laravel blocked it, add this in **_VerifyCsrfToken.php_**

```php
protected $except = [
    '/payment/webhook/maya',
];
```

### Add a Webhook in your routes

```php
  Route::post('/payment/webhook/maya', [CinemaPaymentController::class, 'webhook']);
```

since, signature-based authentication is not yet available in paymaya

```bash
  https://dev-forum.maya.ph/t/webhook-signature-authentication/325/2
```

so we need to validate manually through reference number, add this in your controller

```php
//adjust based on your needs

public function webhook(Request $request)
    {
        $data = json_decode($request->getContent(), true);

        $refNumber = $data['requestReferenceNumber'] ?? null;
        $paymentId = $data['id'] ?? null;
        $status = $data['paymentStatus'] ?? ($data['status'] ?? null);

        if (!$refNumber) {
            return response()->json(['error' => 'missing reference'], 400);
        }

        $order = BookingOrder::where('reference', $refNumber)->first();

        if (!$order) {
            return response()->json(['error' => 'order not found'], 404);
        }

        if ($order->status === 'PAYMENT_SUCCESS') {

            return response()->json(['ok' => true]);
        }

        $updateData = [
            'payment_id' => $paymentId,
            'status' => $status,
        ];

        if ($status === 'PAYMENT_SUCCESS') {
            $updateData['paid_at'] = $order->paid_at ?? now();
        }

        $order->update($updateData);

        return response()->json(['status' => 'ok']);
    }
}
```
